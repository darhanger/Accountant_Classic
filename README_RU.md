**Accountant Classic** — лёгкий экономический журнал, который отслеживает источники получения и направления расходования игровой валюты.

Аддон отдельно показывает доходы и расходы за текущую сессию, день, неделю и всё время наблюдения.

Разработан для **World of Warcraft 3.3.5a**.

</div>

## Возможности

Accountant Classic автоматически отслеживает изменение количества денег персонажа и относит операцию к активности, которая вызвала транзакцию.

Аддон предоставляет:

- автоматический учёт доходов и расходов;
- отдельные суммы поступлений и затрат для каждой категории;
- статистику за сессию, день, неделю и всё время;
- расчёт чистой прибыли или убытка;
- текущую сумму денег у персонажей одного игрового мира;
- автоматический сброс дневного и недельного периодов;
- настройку первого дня недели;
- сброс выбранного периода отчёта;
- перемещаемое окно в стиле стандартного интерфейса Blizzard;
- кнопку у мини-карты с регулируемым положением;
- назначаемую горячую клавишу;
- поддержку панелей LibDataBroker;
- дополнительное обновление кнопки Titan Panel;
- сохранение статистики и настроек между сессиями.

## Категории операций

Изменения денег распределяются по следующим категориям:

| Категория | Примеры |
| --- | --- |
| Обучение | Классовые тренеры и изучение профессий |
| Такси | Оплата перелётов |
| Торговля | Прямой обмен между игроками |
| Аукцион | Покупки, залоги и продажи на аукционе |
| Торговцы | Покупка и продажа предметов у NPC |
| Ремонт | Личная оплата ремонта экипировки |
| Почта | Отправка и получение денег через почту |
| Задания | Награды за задания и связанные операции |
| Добыча | Деньги с существ и распределяемая добыча |
| Прочее | Операции, которые нельзя отнести к другой категории |

Доходы и расходы сохраняются отдельно для каждой категории.

## Периоды отчёта

Главное окно содержит пять вкладок.

### Сессия

Показывает деньги, полученные и потраченные с момента текущего входа в игру.

Значения сессии обнуляются при новой загрузке аддона.

### День

Показывает доходы и расходы за текущий календарный день.

Дневные значения автоматически сбрасываются при смене даты.

### Неделя

Показывает статистику за текущую настроенную неделю.

Первый день недели можно выбрать в настройках аддона.

### Всё время

Показывает все доходы и расходы текущего персонажа с начала наблюдения или с момента ручного сброса общего отчёта.

### Все персонажи

Показывает текущую сумму денег у сохранённых персонажей того же игрового мира и дату их последнего обновления.

Вкладка также рассчитывает общую сумму денег у этих персонажей.

## Главное окно

В окне Accountant Classic отображаются:

- источник операции;
- полученные деньги;
- потраченные деньги;
- общий доход;
- общие расходы;
- чистая прибыль или убыток;
- текущие деньги персонажа;
- дата начала активной недели;
- текущие суммы денег у других персонажей игрового мира.

Окно можно перемещать, удерживая левую кнопку мыши.

## Открытие Accountant Classic

Открыть журнал можно несколькими способами:

- нажать левой кнопкой мыши по кнопке у мини-карты;
- использовать `/accountant`;
- использовать `/acc`;
- назначить действие **Accountant** в стандартном меню горячих клавиш WoW;
- нажать левой кнопкой мыши по значку Accountant Classic на панели LibDataBroker.

Нажатие правой кнопкой мыши по кнопке мини-карты или LibDataBroker открывает настройки аддона.

## Кнопка у мини-карты

Кнопку у мини-карты можно включить или отключить в настройках.

Её положение вокруг мини-карты изменяется ползунком и сохраняется отдельно для персонажа.

| Действие | Результат |
| --- | --- |
| Левая кнопка мыши | Открыть или закрыть окно Accountant Classic |
| Правая кнопка мыши | Открыть настройки аддона |
| Наведение курсора | Показать краткую подсказку |

## Настройки

Accountant Classic добавляет собственный раздел в стандартные настройки интерфейса Blizzard.

Доступные параметры:

- включение или отключение кнопки у мини-карты;
- изменение положения кнопки;
- выбор первого дня недели.

Все параметры сохраняются между игровыми сессиями.

## Команды

Поддерживаются команды `/accountant` и `/acc`.

| Команда | Описание |
| --- | --- |
| `/accountant` | Открыть окно Accountant Classic |
| `/acc` | Открыть окно Accountant Classic |
| `/accountant log` | Открыть окно Accountant Classic |
| `/accountant verbose` | Включить или отключить диагностические сообщения о транзакциях |
| `/accountant week` | Вывести рассчитанную дату начала текущей недели |

Команды `verbose` и `week` в основном предназначены для тестирования и отладки.

## Сброс статистики

Кнопка **Сбросить** очищает выбранный период отчёта для активного персонажа.

Например, можно отдельно сбросить:

- текущую сессию;
- текущий день;
- текущую неделю;
- общую статистику.

Перед удалением данных появляется окно подтверждения.

Вкладка **Все персонажи** через эту кнопку не сбрасывается.

## Установка

1. Скачайте последнюю версию со страницы [Releases](https://github.com/darhanger/Accountant_Classic/releases).
2. Распакуйте загруженный архив.
3. Скопируйте папку `Accountant_Classic` в:

```text
World of Warcraft\Interface\AddOns\
```

4. Убедитесь, что структура папок выглядит примерно так:

```text
World of Warcraft
└── Interface
    └── AddOns
        └── Accountant_Classic
            ├── Accountant_Classic.toc
            ├── Accountant.lua
            ├── Accountant.xml
            ├── AccountantButton.lua
            ├── AccountantButton.xml
            ├── AccountantOptions.lua
            ├── AccountantOptions.xml
            ├── Bindings.xml
            ├── Images
            ├── Libs
            └── Locale
```

5. Перезапустите игровой клиент.
6. Включите **Accountant Classic** в списке модификаций на экране выбора персонажа.

## Использование

После установки Accountant Classic автоматически начинает отслеживать операции с деньгами.

Обычная последовательность работы:

1. Играйте как обычно, получая и расходуя деньги.
2. Откройте Accountant Classic кнопкой у мини-карты или командой `/acc`.
3. Выберите нужный период отчёта.
4. Сравните доходы и расходы по источникам.
5. Посмотрите итоговую чистую прибыль или убыток.
6. Откройте вкладку **Все персонажи**, чтобы увидеть общую сумму денег на текущем игровом мире.

Ручной ввод операций не требуется.

## Сохранение данных

Accountant Classic сохраняет статистику отдельно по:

- игровому миру;
- персонажу;
- категории операции;
- периоду отчёта.

Для сохранения истории и параметров между сессиями используются SavedVariables.

Перед переустановкой игры или удалением папки WTF сохраните следующий файл, если хотите оставить историю:

```text
WTF\Account\<ACCOUNT_NAME>\SavedVariables\Accountant_Classic.lua
```

Точное название папки учётной записи зависит от установки игры.

## Известные ограничения

Accountant Classic определяет источник операции по игровому окну или активности, которая была открыта в момент изменения денег.

Операция может попасть в категорию **Прочее**, если:

- клиент не отправил ожидаемое событие;
- несколько изменений денег произошли в необычном порядке;
- пользовательский сервер изменил стандартную последовательность транзакции;
- другой аддон или серверная система изменили деньги без открытия стандартного окна Blizzard.

Вкладка **Все персонажи** содержит только персонажей текущего игрового мира, которые ранее входили в игру с включённым аддоном.

## Для чего нужен Accountant Classic?

Аддон помогает:

- понять, на что расходуется золото;
- сравнивать доход от заданий, фарма и аукциона;
- отслеживать расходы на ремонт и перелёты;
- рассчитывать прибыль за отдельную игровую сессию;
- анализировать дневной и недельный денежный поток;
- видеть общую сумму денег у нескольких персонажей;
- тестировать изменения экономики на пользовательских серверах;
- проверять транзакции при разработке аддонов или серверного контента.
